## Notes / Caveats
- **REER interpretation thresholds**: neutral (±3%), mild (±7%), meaningful (>±7%).
- **REER methodology note**: The BIS REER uses base year 2020=100, not a rolling 10-year average. The project's "REER vs 10Y" metric is a custom heuristic that calculates deviation from the 10-year historical average for valuation context. This is not official BIS methodology – BIS states REER levels "do not provide information concerning over- or undervaluation."
- The "gap" columns are purely visual separators and do not contain data or formulas.
