## Key Design Decisions

1. **MSCI Links as Raw Text**: No Excel hyperlinks, just URL text. Users can copy/paste if needed.

2. **Definitions After Data**: Matches reference dashboard layout. Definitions start at row 39.

3. **Region as Reference Only**: Removed filtering complexity. Region column (Z) is informational.

4. **Snapshot Date Fixed**: ETF data ends on Feb 27, 2026 to align with MSCI factsheet date. CAGR calculations use actual end date (Feb 27, 2026), not calendar year-end.
